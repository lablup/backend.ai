Add `permissions` and `users` connection fields to `RoleGQL` with Relay-style pagination support.

- `Role.permissions`: Returns `PermissionConnection` with filtering and sorting by permission attributes
- `Role.users`: Returns `RoleAssignmentConnection` with assignment metadata (granted_by, granted_at) and user information
  - Supports filtering by username, email, and granted_by
  - Supports sorting by username, email, and granted_at
  - Access user details via nested `user` field on each assignment node
