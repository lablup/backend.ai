Add a Traefik app-proxy dataplane option to the TUI installer via a new `--proxy-backend=traefik` flag that composes with both `--frontend-mode=port` and `--frontend-mode=wildcard` traffic patterns (`--frontend-mode=traefik` remains as a deprecated alias for the port pattern): Traefik runs as a halfstack compose service (profile `traefik`) with the appproxy Traefik plugin, the installer generates the matching entrypoint set (the portproxy range, or the `domainproxy` wildcard entrypoint), and the coordinator/worker configurations are generated accordingly.
